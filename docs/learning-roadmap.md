# JavaScript Learning Roadmap

A suggested path through this repository based on my learning journey.

## Phase 1: Foundations

Start with the basics — get comfortable with data types, objects, arrays, and functions.

| File | Concepts |
|------|----------|
| `01-basics/01_objects.js` | Object creation, properties, methods, `this`, object merging |
| `01-basics/02_arrays.js` | Array methods, `slice`/`splice`, `flat`, `filter`, `map` |
| `01-basics/03_functions.js` | Functions, closures, IIFE, rest params |
| `01-basics/04_loops_iteration.js` | `for...of`, `for...in`, `Map`, `forEach` |
| `01-basics/05_destructuring_spread.js` | Object/array destructuring, spread, rest |

## Phase 2: DOM & Events

Move to the browser — understand how JavaScript interacts with the DOM.

| File | Concepts |
|------|----------|
| `02-dom/01_todo_app.html` | DOM CRUD, event listeners, state management, filtering |
| `02-dom/02_event_propagation.html` | Event phases, delegation, `stopPropagation`, `preventDefault` |

## Phase 3: Asynchronous JavaScript

Master async patterns — the most critical skill for modern JS.

| File | Concepts |
|------|----------|
| `03-async/01_promises.js` | Promise creation, chaining, `Promise.all`, `Promise.race` |
| `03-async/02_async_await.js` | `async`/`await`, try/catch, sequential async |
| `03-async/03_fetch_api.js` | `fetch()` with `.then` and `async/await` |
| `03-async/04_intervals.html` | `setInterval`, `clearInterval` |
| `03-async/05_xhr_api.html` | XMLHttpRequest, JSON parsing |

## Tips

- Run `.js` files with `node <filename>`
- Open `.html` files directly in a browser
- Use the console (F12) to inspect output
- Modify the code and see what breaks — that's how you learn
