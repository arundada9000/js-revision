# 01 — JavaScript Basics

Core language fundamentals from the ground up. 17 files covering everything from variables to optional chaining.

## Files

| # | File | What You'll Learn |
|---|------|-------------------|
| 00 | `00_variables.js` | `var`, `let`, `const`, hoisting, temporal dead zone |
| 01 | `01_data_types.js` | Primitives vs reference, `typeof`, coercion, truthy/falsy |
| 02 | `02_strings.js` | Template literals, string methods, manipulation |
| 03 | `03_objects.js` | Object literals, properties, methods, `this`, merging |
| 04 | `04_arrays.js` | `slice`/`splice`, `concat`, `flat`, `filter`, `map` |
| 05 | `05_functions.js` | Declarations, expressions, arrow, rest params, IIFE |
| 06 | `06_arrow_functions.js` | Arrow syntax, lexical `this`, limitations |
| 07 | `07_this_keyword.js` | `this` in all contexts |
| 08 | `08_call_apply_bind.js` | Explicit binding, method borrowing |
| 09 | `09_scope_closures.js` | Lexical scope, closures, private variables |
| 10 | `10_loops_iteration.js` | `for...of`, `for...in`, `Map`, `forEach` |
| 11 | `11_destructuring_spread.js` | Destructuring, spread, rest |
| 12 | `12_higher_order_functions.js` | `map`, `filter`, `reduce`, `some`, `every`, `find` |
| 13 | `13_prototypes.js` | Prototype chain, inheritance |
| 14 | `14_classes.js` | ES6 classes, `extends`, `super`, private fields |
| 15 | `15_error_handling.js` | `try`/`catch`/`finally`, `throw`, custom errors |
| 16 | `16_optional_chaining.js` | `?.`, `??`, short-circuit evaluation |

## Run

```bash
node 01-basics/00_variables.js
```
