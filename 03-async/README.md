# 03 - Asynchronous JavaScript

Promises, async/await, fetch API, timers, and legacy XHR.

## Files

| # | File | What You'll Learn |
|---|------|-------------------|
| 1 | `01_promises.js` | Promise creation, `.then`/`.catch` chaining, `Promise.all`, `Promise.race` |
| 2 | `02_async_await.js` | `async`/`await` syntax, `try`/`catch` error handling |
| 3 | `03_fetch_api.js` | Modern HTTP with `fetch()`, `.then` vs `async/await` |
| 4 | `04_intervals.html` | `setInterval` / `clearInterval` with live clock UI |
| 5 | `05_xhr_api.html` | Legacy `XMLHttpRequest` with GitHub API |

## Run

- JS files: `node 03-async/01_promises.js`
- HTML files: open in browser
